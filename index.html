kevin stadler

Originally trained as a software engineer, I received a PhD in Linguistics & English Language from the University of Edinburgh in 2017. Apart from modelling language change I have also worked with (and on) open source software on a a wide range of other topics, in particular cartography and sound.

You can find a lot of the code I write over on Github. I have an irregularly updated blog, and have previously also written for Replicated Typo.

My PhD thesis can be downloaded from the Edinburgh Research Archive. The source code for the models, graphs and statistical analyses can be found in the dedicated github repository.

If you're a linguist or cultural evolution researcher you might be interested in my cultevo R package.

alpinia

alpinia is a Python script wrapped around GDAL's hillshading algorithm whose sole purpose it is to allow easy creation of "comically exaggerated" relief maps. To the left is an example rendition of the mighty mountain range that is home to Greenwich Observatory.
